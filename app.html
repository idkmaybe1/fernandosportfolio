My App Development Journey

I started this journey in my AP Computer Science Principles Class. In this class, we learned how to design apps on Code.org, and how to program functionality using JavaScript. I also learned how to use datasets. I made several apps in this class, some independently, some with partners.

Check out some of my work!



The AP Exam for AP CSP requires that you build an app that fulfills certain requirements. This assignment is called the Create Performance Task (Create PT). For my Create PT, I created an App to help people trying to learn basic phrases in Spanish and Italian.


In my AP CSP course, one of the projects was a hackathon. We were assigned partners and given a few hours of class time to build an app that uses a data set.
