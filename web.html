My Web-Development Journey

What started as something I would do when bored in class turned into a newfound passion. I started this journey towards the end of 11th grade when I started learning HTML and CSS in class. I learned the very basics, but it was enough to initiate a burning curiosity to learn more. To do this, I applied and was accepted to Tech360 - a three-week-long web development boot camp where I gained proficiency with HTML, CSS, and the Bootstrap framework. We created several websites for projects and assignments, and many required collaboration and group work. Our final project was a website for a mock client. After completing this boot camp, I applied and was accepted into Tech Flex Leaders, a year-long program for advanced web development hosted by the same organization: America On Tech. Since then, I have learned how to use APIs and JavaScript.
